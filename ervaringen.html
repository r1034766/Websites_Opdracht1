Ervaringen

In de loop van mijn carrière heb ik aan verschillende projecten kunnen meewerken. Dit heeft ervoor gezorgd dat ik mijn skillset redelijk heb kunnen uitbreiden.

Welke certificaten behaalde ik tijdens mijn Carrière?

TIA Portal

De dag van vandaag is TIA Portal één van de meest gebruikte tools voor het automatiseren in de industriële sector. Deze tool is ontwikkeld door Siemens en wordt onder andere gebruikt om de typische Siemens PLC te programmeren. Deze tool heeft uiteraard nog veel meer functionaliteiten wat het gebuik ervan ook zeer complex maakt. Ik heb via mijn wergever de kans gekregen om de TIA Portal opleiding te volgen.

Deze opleiding bestond uit twee delen. In de basisopleiding leerden we hoe we van nul een automatiserings project konden starten, hoe we een PLC en de juiste modules moesten configureren en tot slot leerden we hoe we effectief programmatie konden maken en downloaden naar de PLC om een simpele testopstelling te laten werken.

In de tweede opleiding gingen wat meer geavanceerde toepassingen van de tool bekijken. Hier bouwden we verder op de kennis vanuit de basis opleiding. We bouwden bijvoorbeeld een meer comlexe testopstelling met "third party" toestellen die we vervolgens met het Siemens materiaal moesten laten werken. zeker niet de gemakkelijkste opdracht!

Aangezien Werken met TIA Portal een zeer gegeerde skill is in de industriële sector ben ik zeer blij dat ik de kans heb gekregen om deze opleiding te volgen.

BA5

Werken aan industriële elektrische installaties kan zeer gevaarlijk zijn. Om het risico op ongevallen te beperken is het daarom zeer belangrijk om goed geïnformeerd te zijn. Alvorens het BA5 certificaat wordt gegeven moet er dus een test worden afgelegd om te bewijzen dat de kandidaad de belagrijkste principes vanuit de elektriciteit onder de knie heeft. Via dit certificaat weet de werkgever dat de technieker de juiste competenties heeft om veilig aan elektrische installaties te werken.

Aan welke projecten heb ik gewerkt?

Klimaatkasten KBC Mechelen

Bij dit project werden een tiental klimaatkasten voor een groote serverroom vervangen door nieuwe. Ook de bestaandesturing van de HVAC installatie moest een upgrade krijgen.

Mijn taak bestond erin om de volledige sturing van de HVAC en de klimaatkasten te vernieuwen. Aangezien alles nog via verouderde technologie gestuurd werd moest ik voor de migratie naar de nieuwe controllers volledig van nul beginnen. Dit was zeker niet vanzelfsprekend aangezien de programmatie van de oude technologie gebaseerd was op code en de nieuwe controllers via functieblokken moesten worden geprogrammeerd.

De integratie van de klimaatkasten op het bestaande gebouw beheer systeem was ook een uitdaging in de zin dat deze niet hetzelfde communicatie protocol gebruikten als het bestaande Siemens materiaal. Om de klimaatkasten toch te kunnen laten communiceren met het bestaande systeem werd er gebruik gemaakt van een integratie via "Modbus". Dit is een standaard communicatie protocol dat compatibel is met een Siemens PLC mits een bijkomende module wordt gebruikt in combinatie met de PLC.

Aangezien ik voor dit project nooit in aanraking ben gekomen met Modbus integraties, heb ik nog snel een on-the-job training gevolgd om zo de nodige kennis op te doen om de verder in dienststelling van de klimaatkasten te kunnen afwerken. Dit is uiteindelijk ook gelukt waardoor ik de klant tevreden heb kunnen achterlaten met een werkende serverroom.
